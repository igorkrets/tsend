# Данные отправляемые на tsend.php
Данные отправляются POST запросом.
## phone
Номер телефона клиента
> phone=+79991112233 (в любом формате)
## email
Email клиента
> email=notify@mail.ru
## name
Имя клиента
## status_id
id этапа сделки
## host
hostname
>host=ya.ru
